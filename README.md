# Number game

The is from an old freeware game of Amiga called "Brain".

## Objective

Try to get similar set of 3 x 3 numbers that is in the Goal grid.

## Rules

If you press unto 0, the square will become an 1 and adjacent squares' value (!== 0) horizontally and vertically will increse by 1. If the value is 4 and the value is to be increased, the value will be set to 1 again.
